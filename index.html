Polarimetric SAR Data Analysis for Land Cover Classification

Open in Full View

This research focuses on the application of polarimetric Synthetic Aperture Radar (SAR) data analysis to classify land cover types in a region near Mount Semeru, East Java. The primary aim is to differentiate between residential areas and adjacent vegetation in the southern part of the region. Using the SNAP software for preprocessing, including steps like orbit update, coregistration, multithreading, and speckle filtering, followed by image classification in ArcMap, the study provides valuable insights into land-use patterns. The findings are useful for environmental monitoring and urban planning in mixed-terrain areas where residential zones and vegetation coexist.
